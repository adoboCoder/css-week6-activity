The Bootstrap v5 grid systems has six (five for v4) of classes: xs (extra small), sm (small), md (medium), lg (large), xl (extra large), and xxl (extra extra large - v5 only). You can use nearly any combination of these classes to create more dynamic and flexible layouts.

Each tier of classes scales up, meaning if you plan on setting the same widths for xs and sm, you only need to specify xs.
